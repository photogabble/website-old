Projects.

In dedicate a varying amount of my spare time each week to working on open source projects. Most notably the static site generator Tapestry and a port of isso to PHP.

Below is a list of other less ambitious projects I have worked on:

Laravel Remember Uploads

A Laravel middleware package written to solve the issue of uploaded files being lost as a side effect of unrelated form validation errors redirecting the user back. It does this by temporarily caching server-side the upload fields that have passed validation so that they may be used once the whole form has been submitted passing validation.

Laravel Registration Validator

A Laravel validation package that acts as a wrapper for my Confusable Homoglyphs library. It provides your application the ability to validate user input as not containing dangerous confusable characters.

PHP Confusable Homoglyphs

A PHP port of vhf/confusable_homoglyphs written after I found myself needing the functionality that the Python version provides.

It allows you to identify if an input string contains confusable homoglyphs – e.g. the Greek letter capital alpha Α looks identical to the Latin capital letter A but are considered as two different homoglyphs.

For more information read this article by James Bennett on validating usernames.

Pixel Editor Tutorial

The source code repository for my tutorial series on writing a pixel editor in JavaScript.

Tagify Vue

A component I wrote for Vue.js that acts as a convenience wrapper for Tagify.

Tuppence

A very small (some might say pico) framework. I threw this together when I noticed that Proton by Alex Bilbie didn't use the (then) latest version of Router by the PHP League.

It brings together a dependency injection container, router and events all provided by The League of Extraordinary Packages.

Tuppence Boilerplate

An opinionated boilerplate that wraps Tuppence to provide a project structure that resembles that used by Laravel. It comes bundled with service providers for the Plates PHP template system and Docrtine ORM.

Clover Coverage Cli

A command line parser of clover.xml code coverage files.
